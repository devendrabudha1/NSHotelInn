BootStrap:

Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites. I have used bootstrap to make our Book Menu. Thanks to Bootstrp framework from which i have used the function of it. They are listed below:

https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css
https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js
https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js

Font Awesome:

Font Awesome is a font and icon toolkit based on CSS and LESS. It was made by Dave Gandy for use with Twitter Bootstrap, and later was incorporated into the BootstrapCDN. It is one of the top open source projects on github.Font Awesome has grown to have over 5,000 icons and continues to add the most popular and needed icons. And i have used this for making the icons of Twitter,Facebook and Instagram. Which is listed below.

https://cdnjs.cloudflare.com/ajax/libs/font-awesome/4.7.0/css/font-awesome.min.css

Google Font API:

Google Font API is a web service that supports high-quality open source font files that can be used on your web designs easily. The font collection will hopefully keep growing to give you the ability to choose many more types of fonts. And i have used this for Playfair Display font. Which is listed below:

https://fonts.googleapis.com/css?family=Playfair+Display:400,400i,700,700i
